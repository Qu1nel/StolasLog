Welcome to StolasLog's Documentation
##################################################

**StolasLog** is an elegant, powerful, and highly configurable logging framework for Python, built on Loguru, Pydantic, and Rich.

This documentation provides a comprehensive guide for users and contributors, from getting started in minutes to designing complex logging configurations.
